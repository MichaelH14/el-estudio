# Recursos libres verificados

> **Cuándo cargar este archivo:** cuando haga falta profundizar en un tema más allá de la base de conocimiento — cada recurso está verificado (URL viva, contenido real, gratis de verdad) y anotado con para qué sirve. Última verificación: 2026-07-18.

## Canon universal (aplica a cualquier motor)

| Recurso | Autor | URL | Qué es y para qué |
|---|---|---|---|
| **Game Programming Patterns** | Robert Nystrom | https://gameprogrammingpatterns.com/ | Libro completo gratis online. EL canon de patrones de arquitectura para juegos: Command, State, Observer, Game Loop, Update Method, Component, Event Queue, Object Pool, etc. Ya destilado en parte en [ver: unity/csharp-patrones]; ir aquí para el detalle de cada patrón |
| **Fast-Paced Multiplayer** | Gabriel Gambetta | https://gabrielgambetta.com/client-server-game-architecture.html | Serie de 4 artículos + demo interactiva: servidor autoritativo, client-side prediction y reconciliación, interpolación de entidades, lag compensation. LA introducción canónica a netcode — fuente semilla obligatoria del tema multijugador de `pipeline/` |
| **Multiplayer Networking Resources** | 0xFA11 (comunidad) | https://github.com/0xFA11/MultiplayerNetworkingResources | Lista curada (CC0) de artículos, charlas GDC, librerías y ejemplos de netcode para Unity/Unreal/C++. Es el repo ORIGINAL (los forks tipo `xuxudefeng/Awesome-Game-Networking` apuntan aquí) |
| **The Nature of Code** (2ª ed.) | Daniel Shiffman | https://natureofcode.com/ | Libro completo gratis (CC): vectores, fuerzas, oscilación, partículas, agentes autónomos, algoritmos genéticos. En p5.js pero los conceptos son agnósticos — para entender la matemática del movimiento antes de implementarla en el motor |
| **CS50's Intro to Game Development** | Harvard (CS50G) | https://cs50.harvard.edu/games/2018/ | Curso completo gratis: 12 semanas reconstruyendo Pong, Flappy Bird, Mario, Zelda, Portal (LÖVE 2D/Lua + Unity/C#). ⚠️ RETIRADO el 2024-06-30 — queda como archivo; sólido para fundamentos, no para APIs actuales |
