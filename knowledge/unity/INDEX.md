# Base de conocimiento Unity — Índice

Cada archivo es autocontenido y arranca con una línea "Cuando cargar este archivo": cargar SOLO los relevantes a la tarea en curso, no toda la base. Las referencias `[ver: slug]` apuntan a otro archivo de esta carpeta; `[ver: gamedev/slug]` apunta a la base hermana de game design (`../gamedev/`), que cubre el QUÉ/POR QUÉ de diseño mientras aquí está el CÓMO en el motor. Para depurar comportamiento raro, empezar por `gotchas-unity`; si hay un editor Unity conectado por MCP, `unity-mcp-flujo` se carga siempre. Todo está verificado contra Unity 6.x + URP (estado a julio 2026).
